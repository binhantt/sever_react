# Server React Project

## Overview
Đây là dự án quản lý hệ thống với giao diện React và backend Node.js/Express, bao gồm các tính năng:
- Quản lý người dùng (thêm/sửa/xóa)
- Quản lý đơn hàng
- Dashboard thống kê
- Hệ thống đăng nhập/đăng xuất

## Key Features
- **Frontend**:
  - React với Vite build tool
  - React Bootstrap cho UI components
  - React Router cho navigation
  - Toast notifications
  - Form validation
  - Responsive design

- **Backend**:
  - Node.js với Express framework
  - RESTful API endpoints
  - JWT Authentication
  - Database integration (MongoDB/MySQL)

## Development Setup
Công nghệ sử dụng:
- React 18
- Vite
- React Bootstrap 5
- React Icons
- React Toastify
- ESLint cho code quality

Các plugins chính:
- @vitejs/plugin-react
- eslint-plugin-react-hooks
- eslint-plugin-react-refresh

## Cấu trúc thư mục chính
- `src/components`: Chứa các UI components
- `src/pages`: Chứa các page components
- `src/config`: Cấu hình và dữ liệu mẫu
- `src/main.jsx`: Entry point

## Getting Started
1. Clone repository: `git clone [repo-url]`
2. Cài đặt dependencies: `npm install`
3. Chạy development server: `npm run dev`
4. Build production: `npm run build`
5. Chạy production: `npm run preview`
